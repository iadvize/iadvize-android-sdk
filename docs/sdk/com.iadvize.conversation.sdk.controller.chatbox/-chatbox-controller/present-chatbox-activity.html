Display the Conversation View as a Activity (will appear on top of your current view hierarchy).

By default, if the Chat Button is displayed and the user tap on it, we will show the Conversation View Activity automatically.

You can use this method to display manually the Conversation View to the user (if you don't want to use the Chat Button behavior and use, for example, a custom element in a view to allow the user to enter in the Conversation View).

Please visit https://github.com/iadvize/iadvize-android-sdk for further information.
