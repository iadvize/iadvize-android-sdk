Parameters

mainColor

Main color of the Conversation View, in hexadecimal,

e.g.: 0x0099FFFF, ContextCompat.getColor(context, R.color.sdk_color), Color.YELLOW, etc..

The main color will be used, in the Chat Button, for:

  • The color of the default Chat Button (if not overrided)

in the Conversation View, for:

  • The color of the outgoing message bubble

  • The color of the send button in the message composition bar

  • The color of the links in the incoming messages

toolbarTitle

Title in the Toolbar bar of the Conversation View.

Should be localized.

automaticMessage

Content of the first automatic message that invites your user to start asking a question.

Should be localized.

By default, this property is null and no automatic message will be displayed in the Conversation View.

gdprMessage

Content of the first GDPR message which invites your user to accept or decline the GDPR consent.

Should be localized.

By default, we will use a default GDPR consent message (if the GDPR feature is activated).

fontPath

Path of the font file. The font file should be included in the assets folder within the App (e.g.: /fonts/helvetica-bold.ttf).

toolbarBackgroundColor

Background color of the Navigation Bar of the Conversation View.

By setting this property, you will update the background color of the Navigation Bar so the Conversation view will fit your navigation experience.

By default, we will use a white color.

toolbarMainColor

Color of the elements which are contained in the Navigation Bar.

By setting this property, you will update the color of all the elements which are in the Navigation Bar of the Conversation View: buttons and labels.

By default, we will use a black color.

incomingMessageAvatar

Avatar of the operator.

By setting this property, you will update the avatar displayed for the incoming messages. You can set an Url or a local image.

Please visit https://github.com/iadvize/iadvize-android-sdk for further information.
