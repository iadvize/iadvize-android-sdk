logLevel

@JvmStatic
var logLevel: Logger.Level

Level of the iAdvize Conversation SDK console logger.

By default, the iAdvize Conversation SDK will only log message of WARNING and ERROR level. You can change this value to have more feedbacks on the iAdvize Conversation SDK.

N.B. The logger will only print logs in DEBUG mode.
